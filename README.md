osx-direnv
==========

An Ansible role to setup direnv on OS X

Requirements
------------

[Homebrew](http://brew.sh/) must be installed.

Role Variables
--------------

None.

Dependencies
------------

- hnakamur.oh-my-zsh

Example Playbook
----------------

    - hosts: servers
      roles:
         - hnakamur.osx-direnv
